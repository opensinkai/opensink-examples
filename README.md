# OpenSink Examples

Example agents built with [OpenSink](https://opensink.com) — the control layer for AI agents in production.

## Examples

| # | Example | Description |
|---|---------|-------------|
| 1 | [Finance News Agent](examples/1-finance-news-agent-fastify) | Fetches financial news, analyzes with GPT-4o-mini, and stores results in a sink. Demonstrates sessions, configs, activities, and sink items. |
| 2 | [Trading Agent](examples/2-trading-agent-fastify) | Proposes trades based on news sentiment, waits for human approval, then executes. Demonstrates human-in-the-loop with input requests and execution endpoints. |

Each example is a standalone Fastify app with its own setup instructions.
